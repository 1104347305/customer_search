## 如何重新启用

如需重新启用数据脱敏，修改 `app/services/search_service.py` 第55-63行：

```python
# 数据脱敏
masked_documents = self.data_masking.mask_customers(result["documents"])

# 构建响应（符合V3接口规范）
data_response = SearchDataResponse(
    total=result["total"],
    page=page,
    size=size,
    list=masked_documents  # 返回脱敏后的数据
)
```

然后重启API服务。

## 安全提示

⚠️ **重要**: 关闭数据脱敏后，API将返回完整的敏感信息，请确保：
1. API访问有适当的权限控制
2. 网络传输使用HTTPS加密
3. 日志中不记录敏感信息
4. 前端展示时考虑是否需要脱敏
