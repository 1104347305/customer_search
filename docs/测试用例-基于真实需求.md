#### 用例1.1: 查找45岁以上未配置养老保险的客户

**业务场景**: 养老保险产品推广
**查询条件**:
```json
{
  "header": {"agent_id": "A000001", "page": 1, "size": 50},
  "query_logic": "AND",
  "conditions": [
    {"field": "age", "operator": "GTE", "value": 45},
    {"field": "held_product_category", "operator": "NOT_CONTAINS", "value": "养老保险"}
  ],
  "sort": [{"field": "age", "order": "desc"}]
}
```
**预期结果**: 返回45岁及以上且未持有养老保险的客户列表
